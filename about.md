# 关于这个博客

这里会记录我在编程、AI、工具链和日常思考中的一些内容。

## 我会写什么

- 工程实践与踩坑记录
- 对新工具的试用体验
- 适合以后回看的学习笔记

## 如何发布新文章

只需要在 `_posts/` 目录下新增一个 Markdown 文件，命名格式如下：

```text
YYYY-MM-DD-article-title.md
```

然后在文件顶部添加 Front Matter：

```yaml
---
title: 文章标题
date: 2026-04-15
tags: [GitHub Pages, Markdown]
---
```

正文直接写 Markdown 即可，推送到 GitHub 后会自动出现在首页。
